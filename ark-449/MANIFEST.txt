     superposition (P(Q_P=1) ~ 0.50), which could never meet C1
     (S_A_min >= 0.90) — a guaranteed FAIL by construction. Corrected
     to drive the payload to |1> so ALLOW retention can pass.
  2. Payload-bit parser split()[-1] -> split()[0]. Registers are
     (c_auth, c_state, c_pay); Qiskit prints them reverse order, so
     c_pay is the leftmost field. v1.0 read c_auth (always 1).

  formatting crash in the analysis script.

  ALLOW = 1.0000, all seven DENY arms = 0.0000, Delta_B = 1.00.
